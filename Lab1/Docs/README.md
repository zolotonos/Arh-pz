# Lab 1: CRUD Baseline

REST API для системи обліку колекції з базовою JWT-автентифікацією.

## Вимоги

* Node.js (v18+)
* npm

## Встановлення та запуск

1. Встановіть залежності:
   ```
   npm install
   ```
2. Запустіть сервер:
   ```
   node index.js
   ```

Сервер буде доступний за адресою `http://localhost:3000`.

## Тестування

Проєкт покритий Unit та Integration тестами за допомогою Jest та Supertest.
Для запуску тестів виконайте:

```
npm test
```
